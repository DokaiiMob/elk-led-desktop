# ELK LED Desktop

Десктопное приложение для управления Bluetooth LED-лентой **ELK-BLEDOM** на Windows, macOS и Linux.

Переписано с Python/Tkinter на **Electron + React + Node.js + TypeScript**.

## Возможности

- **BLE-подключение** — выбор устройства ELK через системный диалог Bluetooth
- **Быстрое переподключение** — к последнему устройству без повторного выбора
- **Быстрые пресеты** — 10 готовых цветов одним нажатием
- **Встроенные эффекты ленты** — прыжки, градиенты, стробоскоп (2 протокола)
- **Тёплый / холодный белый** — режим цветовой температуры
- **Софт-анимации** — дыхание, стробоскоп, «полиция», свеча
- **Сохранённые сцены** — до 6 избранных цветов с яркостью
- **Режим музыки** — яркость в такт басу + индикатор уровня
- **Радуга** — плавная смена цвета; опция «радуга + музыка»
- **Ручное управление** — цвет, яркость, включение и выключение
- **Остановить все режимы** — одна кнопка для сброса

## Требования

- [Node.js](https://nodejs.org/) 20+
- Bluetooth-адаптер с поддержкой BLE
- LED-лента ELK-BLEDOM (или совместимая по протоколу)

На Windows для BLE может потребоваться установка драйверов Bluetooth и прав администратора при первом запуске.

## Установка

```powershell
git clone https://github.com/DokaiiMob/elk-led-desktop.git
cd elk-led-desktop
npm install
```

## Запуск в режиме разработки

```powershell
npm run dev
```

## Сборка

```powershell
npm run build
npm run dist
```

Готовый установщик появится в папке `release/`.

## Использование

1. Включите ленту и Bluetooth на компьютере.
2. Нажмите **Подключиться** и выберите устройство ELK в системном диалоге.
3. Выберите режим:
   - **Музыка в такт** — выберите источник звука (микрофон или системный звук через захват экрана)
   - **Радуга** — автоматическая смена цветов
   - **Ручное управление** — задайте цвет и яркость вручную

### Звук для режима музыки

Приложение использует **стандартный микрофонный вход** (Web Audio API), без WASAPI:

| Источник | Описание |
|----------|----------|
| **Стерео микшер** | Звук с ПК напрямую (включите в Windows: Параметры звука → Запись → Стерео микшер) |
| **Микрофон** | Микрофон у колонок — универсальный вариант |
| **VB-Audio / Voicemeeter** | Виртуальный аудиокабель, если установлен |

Нажмите **Обновить список устройств** на вкладке «Музыка», если только что включили Стерео микшер.
